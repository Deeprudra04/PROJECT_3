# IRoC-U 2025 – ISRO Robotics Challenge Project

## Overview
This repository contains our team's work for the **ISRO Robotics Challenge – University Edition 2025 (IRoC-U 2025)** organized by the **U R Rao Satellite Centre (URSC), ISRO**.

The challenge focuses on developing **autonomous navigation capabilities for aerial vehicles operating in GPS-denied environments**, inspired by future Mars exploration missions.

Our team successfully **qualified for Phase-1 (Preliminary Round)** of the competition.

---

## Problem Statement
The objective of the challenge is to design and demonstrate an **Autonomous Navigation for an Aerial Vehicle (ANAV)** system capable of operating **without external navigation aids such as GNSS or pseudolite systems**.

The aerial system must be capable of:

- Autonomous navigation in GPS-denied environments
- Stable vertical take-off and landing
- Scanning the environment to detect safe landing zones
- Landing at multiple safe points
- Returning safely to the home position

---

## Project Objectives

- Develop algorithms for **autonomous aerial navigation**
- Implement **vision and sensor-based navigation**
- Simulate **Mars-like terrain exploration**
- Design **safe landing zone detection algorithms**
